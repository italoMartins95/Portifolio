Sobre mim

Java foi a primeira linguagem de programação que eu aprendi. Estudar Java assim de cara foi bem difícil pois essa é uma linguagem fortemente tipada e tem uma sintaxe bem extença. Mas os estudos abriram minha mente para entender vários conceitos importantes de POO, tais como:

  • Classes
  • Objetos
  • Instanciamento
  • Atributos & Metodos
  • Tipos primitivos de dados
  • Visibilidade
  • Herança
  • Polimorfismo
  • Interfaces
  • Abstração

Em seguida passei a estudar HTML5 e CSS3 e comecei a desenvolver minhas primeiras páginas. Aprender essas dus linguagens de marcação foi muito divertido pra mim, então decidir me tornar um Dev FrontEnd e pra isso eu não poderia deixar de lado o tão famoso JavaScript.

Me apaixonei por essa linguagem e desde então tenho aprendido cada vez mais a usar essa ferramenta. Atualmente voltei minha atenção a uma biblioteca poderosa chamada React, onde estou construindo meus primeiros projetos.

Meu objetivo a curto prazo dominar tacnologias que façam de mim um excelente Desenvolvedor FrontEnd e a longo prazo pretendo desenvolver habilidades que me façam ir alem do que os olhos dos usuários podem ver, quero ser um Desevolvedor FullStack e dominar outras tecnologias como:

  • Java
  • SQL
  • NoSQL
  • AWS
  • ReactNative
  • Python
